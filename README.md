# Easy Markdown 编辑器

一个简洁易用的 Markdown 编辑器，专为 macOS 设计，支持实时预览和打印功能。

## ✨ 主要特性

- 📝 **实时编辑**：流畅的编辑体验，支持 Tab 键缩进
- 👁️ **实时预览**：边写边看，即时查看 Markdown 渲染效果
- 🎨 **语法高亮**：支持多种编程语言的代码语法高亮
- 🖨️ **打印支持**：完美适配打印机，支持打印预览
- 💾 **文件操作**：新建、打开、保存 Markdown 文件
- ⚡ **快捷键**：丰富的快捷键支持，提升编辑效率
- 📊 **实时统计**：显示行数、字数、字符数统计
- 🔄 **多视图模式**：编辑器、预览、双栏三种视图模式切换

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 运行应用

```bash
npm start
```

## ⌨️ 快捷键

### 文件操作
- `Cmd+N` - 新建文件
- `Cmd+O` - 打开文件
- `Cmd+S` - 保存文件
- `Cmd+Shift+S` - 另存为
- `Cmd+P` - 打印文档
- `Cmd+Q` - 退出应用

### 编辑操作
- `Cmd+Z` - 撤销
- `Cmd+Shift+Z` - 重做
- `Cmd+X` - 剪切
- `Cmd+C` - 复制
- `Cmd+V` - 粘贴
- `Cmd+A` - 全选
- `Tab` - 插入 4 个空格

### 视图操作
- `Cmd+E` - 切换预览模式（双栏 → 预览 → 编辑器 → 双栏）
- `Cmd+R` - 重新加载
- `Cmd+Option+I` - 开发者工具
- `Cmd+0` - 重置缩放
- `Cmd++` - 放大
- `Cmd+-` - 缩小

## 📋 功能说明

### 编辑器
- 支持所有标准 Markdown 语法
- 单色编辑器界面，减少干扰
- Tab 键自动转换为 4 个空格
- 支持大文件编辑

### 预览器
- 实时渲染 Markdown
- 支持 GitHub Flavored Markdown (GFM)
- 代码块语法高亮
- 表格、列表、引用等完整支持

### 打印功能
- 使用 `Cmd+P` 或点击打印按钮
- 自动格式化打印内容
- 保留 Markdown 样式
- 支持所有标准打印机

### 视图模式
1. **双栏模式**：左侧编辑，右侧预览
2. **预览模式**：全屏预览渲染效果
3. **编辑模式**：全屏编辑，专注写作

## 📦 项目结构

```
easyMarkdown/
├── main.js          # Electron 主进程
├── index.html       # 主界面
├── renderer.js      # 渲染进程脚本
├── styles.css       # 样式文件
├── package.json     # 项目配置
└── README.md        # 说明文档
```

## 🛠️ 技术栈

- **Electron** - 跨平台桌面应用框架
- **Marked.js** - Markdown 解析器
- **Highlight.js** - 代码语法高亮
- **原生 JavaScript** - 无额外框架依赖

## 📝 Markdown 支持

支持所有标准 Markdown 语法，包括但不限于：

- 标题（H1-H6）
- 粗体、斜体、删除线
- 有序列表、无序列表
- 链接、图片
- 代码块、行内代码
- 表格
- 引用
- 分隔线
- 任务列表
